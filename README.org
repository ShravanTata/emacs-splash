*A less intrusive splash screen for GNU Emacs*

[[./splash.png]]

*Features*

- Text is short and centered
- No logo (not even in graphical mode)
- Vertical and horizontal scroll bars are hidden
- Modeline is hidden
- " ", "x", "q", <esc>,r <return> or <mouse-1> kills the buffer
- "C-h" shows the regular splash screen
- Screen is automatically killed after 5 seconds
